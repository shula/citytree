Captchas Blow is a Django + newforms based Captcha system

It's great because:

  • It's very configurable
  • It generates valid xhtml 1.0
  • It's extremely easy to use with an existing newform

I've released three versions of Captchas Blow so far, but I really don't know what exactly changed between them. 0.1 was the first version that I used on my production site 0.2 only has minor cleanups as far as I know.
